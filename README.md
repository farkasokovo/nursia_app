# nursia_app
# Contexto de Desarrollo: Nursia

## 👤 Perfil del Desarrollador
- **Nombre:** Diego Muñoz
- **Ubicación:** México
- **Stack:** Flutter / Dart / Python
- **Entorno:** Ubuntu Linux (ThinkPad T14 Gen 2, 16GB RAM)
- **Método de Debug:** Dispositivo físico Android (no emuladores).

## 🎯 Objetivo del Proyecto
Desarrollar una herramienta clínica integral para enfermería en México. Nursia busca digitalizar cálculos y escalas siguiendo estándares oficiales (como la NOM-045-SSA2-2005) y fuentes científicas (AEMPS, Vademecum).

## 🛠 Especificaciones Técnicas
- **Arquitectura:** Local-first (100% Offline).
- **Base de Datos:** SQLite / Hive (sin servidores externos por ahora).

## 🧠 Instrucciones para la IA
1. **Precisión Médica:** Todas las sugerencias deben usar terminología clínica correcta en español.
2. **Rendimiento:** El código debe estar optimizado para dispositivos físicos, evitando consumo excesivo de RAM.
3. **Privacidad y Seguridad:** Al sugerir mejoras, priorizar el cifrado local de datos sensibles del paciente.
4. **Contexto Regional:** Considerar siempre el contexto del sistema de salud en México.
